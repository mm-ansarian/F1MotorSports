# F1 MotorSports
![A beautiful view of our website's main page.](F1MotorSports.png)

This is a website about the Formula one sports.

As the first ideas, we cover news about F1, write articles and serve you some beautiful movies and series to watch about Formula 1.

## Used tools
- **Backend**
    - ***Programming language:*** Python
    - ***Framework:*** Django
    - ***Database:*** SQLite (for know)

- **Frontend**
    - HTML
    - CSS
    - Java Script
